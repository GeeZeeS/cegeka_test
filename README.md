# cegekaTest



## To build and start the project
```
docker-compose up -d --build
docker-compose up
```

# Contents
To access web contents pages go to: http://127.0.0.1:5000

Available pages:
* /index
* /personal
* /experience
* /education


# Shell Contents
To access shell contents:
```
docker exec -it cegeka bash
python manage.py output_cv
```
