# cegekaTest



## To build and start the project
```
docker-compose up -d --build
docker-compose up
```

# Contents
To access contents vie web pages go to: http://localhost:5000

Available pages:
* /index
* /personal
* /experience
* /education


# Shell Contents
To access contents via shell:
```
docker exec -it cegeka bash
python manage.py output_cv
```
